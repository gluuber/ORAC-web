Help

The management forms above allow you to update each of the data sets and associated pages on the NSW ORAC web site. This is done by loading a CSV file containing the data which is then processed by a server-side script.

It is important that the format of the CSV data follows the examples below. Do not add notes or comments or other extraneous text.
