Data tables CSV files

It is important that the format of the Comma Separated Values (CSV) data follows the examples below. Do not add notes or comments or other extraneous text. The first line should contain the column titles as shown and in the order as shown. Remaining rows should only contain data. Escape commas in text by surrounding values with quotes.

Use Excel or Google Sheets to download/export a CSV file from your spreadsheet of choice.
